# Pouya's resume template

This is a simple SvelteKit template that reads resume data from a YAML file and makes a resume template.

## Features
- Minimal
- Fast
- Responsive
- Printer friendly
- ATS friendly
- 100 Lighthouse score
- No JS

## How to use
```
npm install
npm run dev
```
Edit `src/resume.yml` file and put your own data
